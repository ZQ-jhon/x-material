# x-material

## INTRO

此项目包含对 @angular/material 基础组件二次封装，以及针对项目需求开发的公共组件，各组件的使用方法见 demo 文件夹。

你可以将 demo 运行后在本地查看各组件的实际效果。

## DEMO

```bash
npm start
```

## PUBLISH

1. cd projects/x-material
2. change the code
3. npm version patch
4. ng build x-material
5. cd dist/x-material
6. npm login
7. npm publish [--tag beta]
